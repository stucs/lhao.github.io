JDBC

初始时使用JDBC访问数据库.比较麻烦,访问数据库分为以下几个步骤:

  1. 向DriverManager注册驱动器
  2. 建立与数据库的链接 con = java.sql.DriverManager.getConnection(url, user, pass)
  3. 创建State对象,准备执行SQL语句 Statement stat = con.createStatement()
  4. 执行SQL语句 ResultSet set = stat.executeQuery(“sql sentence”)
  5. 访问set中的结果集
  6. 依次关闭Resultset Statement Connection

使用SessionFactory

使用模板解决了重复代码的问题,但需要手动建立对象和数据库表之间的映射关系.ORM解决了这个问题,常用的ORM框架有Hibernate\Mybaits

添加注解

  • 在Repository类或方法上需要添加@Transactional注解;否则报错:
    Could not obtain transaction-synchronized Session for current thread;
    nested exception is org.hibernate.HibernateException:
    Could not obtain transaction-synchronized Session for current thread

  • 实体需要添加@Entity和@Table注解,并注明primery key:

  • RootConfig需要添加@EnableTransactionManagement,并注册HibernateTransactionManager bean.
